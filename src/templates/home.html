Tubes Algeo 2: Content-Based Information Retrieval

Konsep Search Engine ini

Content-Based Image Retrieval (CBIR) adalah sebuah proses yang digunakan untuk mencari dan mengambil gambar berdasarkan kontennya. Proses dimulai dari mengekstrak gambar query dan dataset untuk mendapatkan fitur-fitur utamanya seperti warna, tekstur, bentuk, dll. Setelah didapatkan fitur, CBIR melakukan algoritma pencocokan untuk membandingkan fitur-fitur yang ada. Fitur yang memiliki tingkat kecocokan paling tinggi dengan fitur gambar query akan ditampilkan, sehingga muncul gambar-gambar yang "mirip" dengan query berdasarkan fitur tertentu.

Search Engine ini memanfaatkan teknik Content-Based Image Retrieval (CBIR) untuk mengidentifikasi gambar berdasarkan konten visualnya, seperti warna dan tekstur. Program akan kemudian menggunakan informasi yang didapatkan untuk mencari gambar di dalam database yang menyerupai gambar awal.

Tahap Pemakaian

  1. Upload folder dataset yang hendak digunakan pada bagian input folder dataset.
  2. Upload gambar query yang ingin dilakukan pencarian di dalam dataset.
  3. Pilih metode CBIR yang ingin digunakan (warna/tekstur).
  4. Klik tombol submit dan silahkan tunggu hasilnya.
